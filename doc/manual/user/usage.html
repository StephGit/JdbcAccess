JdbcAccess - Users' Manual - Usage

Using the Binaries of the Project

This JDBC driver for MS Access is to be used like any other JDBC driver. It is recommended, that the static method AccessDriver.register() is used, to register it with the DriverManager.

The support for executing SQL is based on a parser for SQL:2008 and is limited to the single-table query functionality needed for SIARD Suite 2.1.

Top of page

JDBC URI

This is the format of a JDBC URI for JdbcAccess:

jdbc:access://<host>/<database file name URL-encoded for UTF-8>

E.g.

jdbc:access://localhost/D:/Projekte/JdbcAccess/testfiles/testempty.accdb

or (relative to D:\Projekte\JdbcAccess/)

jdbc:access:testempty.accdb

A GUI for JdbcAccess

JdbcAccess is distributed as a collection of JAR files without a graphical user interface. However, the versatile free application SQuirreL SQL Client can be used to make use of it.
