JdbcAccess - Users' Manual - Purpose

Purpose of the Project

JdbcAccess was developed as part of the SIARD project. It implements the JDBC interface for MS Access databases avoiding the JDBC-ODBC bridge. Therefore it can be used on any (Windows or non-Windows) platform where JAVA is installed. It makes use of Jackcess project for reading and writing relational data in MS Access database files. It presents a standardized (JDBC 4.1, SQL:2008) JDBC interface. However, it only supports single-table queries and has only been tested to the extent, that was used in the SiardCmd project.
